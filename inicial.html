Tema

Tema sobre as aulas de HTML & CSS

Criar um site em html com o tema de uma prefeitura.

    Orientações:
  1. Neste site devem conter cinco páginas: inicial, sobre, locais, dados e fale conosco.
  2. Deve existir um menu com links para as cinco páginas.
  3. A página inicial deve conter eventos da cidade (inventar).
  4. A página sobre deve conter um texto sobre a prefeitura (pode ser o texto do site da prefeitura natal de cada um ou pode-se inventar um texto), acrescido de uma foto.
  5. A página locais devem conter informações, links (se disponível) e imagens sobre locais importantes da cidade (escolas, monumentos, igrejas, etc.)
  6. A página dados deve conter uma tabela contendo dados da cidade (número de habitantes, densidade demográfica, etc.)
  7. A página fale conosco deve conter um formulário de contato com a prefeitura (mínimo de 5 campos).
  8. Deve-se formatar o site aplicando comandos CSS.

      9. Orientações:
    1. As cinco páginas devem ser cinco arquivos html diferentes
    2. Escrever o código css em arquivos separados e efetuar a importação do código para as respectivas páginas
    3. Evitem copiar código dos exercícios que vocês fizeram para o projeto, tentem escrever tudo, para melhor fixação do que cada tag faz.
    4. Procure identar certinho todo o código.
    5. Para a tabela, não esqueça de utilizar todas as tags existentes que uma tabela precisa
    6. Para o formulário, não esqueça também de adicionar labels, tente também dividir o formulário com fieldsets.
    7. Quando finalizarem tudo, encaminhem para eu dar uma olhada nos códigos de vocês e verificar o que preciso reforçar. Encaminhem para
